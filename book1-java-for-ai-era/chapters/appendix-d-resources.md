## Books

### For Beginners
1. **Head First Java** by Kathy Sierra & Bert Bates
   - Visual, engaging introduction to Java
   - Great for absolute beginners

2. **Core Java Volume I** by Cay S. Horstmann
   - Comprehensive reference
   - Updated for modern Java

### For Intermediate
3. **Effective Java (3rd Edition)** by Joshua Bloch
   - Best practices and idioms
   - Essential for writing quality code

4. **Java Concurrency in Practice** by Brian Goetz
   - Deep dive into concurrent programming
   - Industry standard reference

### For Advanced
5. **Java Performance** by Scott Oaks
   - Performance tuning and optimization
   - JVM internals

6. **Modern Java in Action** by Raoul-Gabriel Urma
   - Streams, lambdas, modern features
   - Practical examples

## Staying Current

### What's New in Java
- Follow OpenJDK JEPs
- Subscribe to Java Magazine
- Watch conference talks

### Release Schedule
Java releases every 6 months (March and September):
- LTS versions: 8, 11, 17, 21, 25...
- Feature releases: Everything else

### Features by Version
| Version | Key Features |
|---------|--------------|
| Java 8 | Lambdas, Streams, Optional |
| Java 11 | var, HttpClient, String methods |
| Java 14 | Switch expressions, Records preview |
| Java 16 | Records, Pattern matching |
| Java 17 | Sealed classes, Pattern matching |
| Java 21 | Virtual threads, Pattern matching |

## Creating Your Learning Path

### Beginner (0-6 months)
1. Complete this book
2. Build 2-3 small projects
3. Practice on coding platforms
4. Join r/learnjava community

### Intermediate (6-12 months)
1. Read "Effective Java"
2. Learn Spring Boot basics
3. Contribute to open source
4. Build a full-stack application

### Advanced (1+ years)
1. Deep dive into concurrency
2. Study JVM internals
3. Learn performance optimization
4. Consider certification

*Remember: The best resource is writing code. All the reading in the world won't replace practice. Build things!*
